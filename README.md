# Fimatic APP
## Objetivo y funcionamiento de la aplicación:
La aplicación fue diseñada y creada con el objetivo de enseñar lógica a niños de entre 8 y 14 años, usando bloques que se acoplan para crear una secuencia de instrucciones que son las acciones que realizará el vehículo,
no solo dentro de la app, sino que tienen la posibilidad de vincular un pequeño carro real, controlado por un arduino, que replica los movimientos indicados. 
De esta forma se le agrega un sentiminento de que se está "programando" en la vida real, así motivar al aprendizaje de intrucciones cada vez más complejas.  
Por otro lado, cabe agregar de que esta aplicación fue creada únicamente por mí Nicolás Leiton, para la asignatura de Aplicaciones móviles en 2024.

## Herramientas de desarrollo:
- Figma: Para diseñar el mockup de la aplicación.
- Android Studio: IDE dónde se creó toda la aplicación.
- Kotlin: Lenguaje de programación usado en el desarrollo.
- Arduino/C: IDE y lenguaje usado para la implementación de un dispositivo físico.
